## 📌 概述

腾讯云轻量应用服务器 Lighthouse 提供可视化配置面板，几步即可完成 OpenClaw 接入钉钉机器人。

---

## 📋 前置准备

### 必备条件
- ✅ 具有企业管理员权限的钉钉账号
- ✅ 已部署 OpenClaw 的 Lighthouse 服务器
- ✅ 如果早于 2026-02-11 部署，需先更新版本

### 推荐配置
- 国内服务器
- 国内模型服务（如 GLM-5）

---

## 🔧 配置步骤

### 第一步：创建钉钉应用

1. 前往 [钉钉开发者平台](https://open-dev.dingtalk.com)
2. 登录管理员账号
3. 进入「应用开发 - 钉钉应用」
4. 点击「创建应用」
5. 填写应用名称和描述

---

### 第二步：添加机器人能力

1. 选择创建的应用
2. 点击「机器人」
3. 点击「添加」

---

### 第三步：配置机器人

1. 填写所有必填项
2. 可选项保留默认值
3. 点击「发布」
4. 点击「查看版本详情」
5. 点击「编辑」图标
6. 配置应用可见范围（可选）
7. 点击「保存」→「确认发布」

---

### 第四步：获取凭证

进入「凭证与基础信息」页面，复制：
- **Client ID**
- **Client Secret**

⚠️ **注意保密，勿泄露！**

---

### 第五步：配置 Lighthouse

#### 进入控制台
1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/lighthouse)
2. 选择已部署 OpenClaw 的服务器
3. 进入「管理实例」页

#### 配置模型
在「应用管理」页面配置模型 API Key

#### 配置通道
1. 进入「应用管理」页签
2. 在「通道配置」选择「钉钉」
3. 输入 Client ID 和 Client Secret
4. 点击「添加并应用」
5. 等待显示「运行中」

---

## 💬 使用方式

### 私聊场景
1. 在钉钉搜索机器人应用名称
2. 添加后开始对话

### 群聊场景
1. 进入群聊 → 群设置
2. 点击「机器人」→「添加机器人」
3. 搜索并添加机器人
4. @机器人 进行对话

## ⚠️ 注意事项

1. **模型能力限制**
   - 钉钉仅作为消息通道
   - 实际能力取决于配置的模型

2. **图片/文件处理**
   - 需要配置支持图片理解的模型才能分析图片
   - 需要配置支持图片生成的模型才能发送图片

3. **多通道支持**
   - OpenClaw 支持同时接入多个通道
   - 如：钉钉 + 飞书可同时运行

## 🛠️ 常见问题

### HTTP 401: invalid access token
- 检查 Client ID 和 Client Secret 是否正确
- 确认凭证未过期

### 机器人无响应
- 检查通道状态是否为「运行中」
- 验证模型 API Key 是否有效
- 查看 Lighthouse 日志排查问题

### 钉钉发布需要审批
- 需要联系企业管理员审核
- 或使用个人测试账号创建应用
