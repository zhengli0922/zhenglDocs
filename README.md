# 个人知识库

欢迎来到我的个人知识库！

## 📁 目录结构

| 文件夹 | 用途 |
|--------|------|
| **股市日记** | 股市交易记录与分析 |
| **Projects** | 项目笔记 - 有明确目标和时间范围的任务 |
| **Videos** | 视频笔记 - 视频内容整理与总结 |
| **02-Areas** | 领域笔记 - 持续维护的主题（编程、理财、工作） |
| **03-Resources** | 资源笔记 - 参考资料、读书笔记 |
| **Templates** | 模板 - 快速创建新笔记 |

## 🚀 快速开始

1. **股市记录** → `股市日记/`
2. **项目文档** → `Projects/`
3. **视频笔记** → `Videos/`
4. **技能学习** → `02-Areas/`
5. **读书笔记** → `03-Resources/`

## 📝 模板使用

- `Ctrl/Cmd + N` 创建新笔记
- 使用 Templates 文件夹中的模板

## 🔗 双向链接

- `[[笔记名]]` - 链接到其他笔记
- `#标签` - 添加标签
- `![[笔记名]]` - 嵌入其他笔记
