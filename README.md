## 分类规则（AI 友好）

### 股市日记 `/股市日记/`
- **存放内容**：A股交易记录、复盘、市场分析
- **触发条件**：包含股市、股票、大盘、板块、涨跌等关键词
- **命名格式**：`YYYY-MM-DD-主题.md`
- **Frontmatter**：
  ```yaml
  ---
  title: 标题
  date: YYYY-MM-DD
  tags: [A股, 开盘总结, 上证指数]
  type: daily-stock
  ---
  ```

### 项目笔记 `/Projects/`
- **存放内容**：有明确目标、时间范围的项目文档
- **触发条件**：项目名、功能开发、特定任务
- **命名格式**：`项目名-功能描述.md` 或 `项目名/子文档.md`
- **特点**：有开始和结束时间

### 视频笔记 `/Videos/`
- **存放内容**：视频/直播/课程的内容整理
- **触发条件**：来源是视频平台（抖音、B站、YouTube等）
- **命名格式**：`视频主题-作者或来源.md`
- **Frontmatter**：
  ```yaml
  ---
  title: 标题
  date: YYYY-MM-DD
  source: 视频链接
  author: 作者
  type: video
  tags: [标签1, 标签2]
  ---
  ```

### 领域笔记 `/02-Areas/`
- **存放内容**：持续维护、没有明确结束时间的主题
- **子目录**：
  - `编程/` - 编程技术、工具使用、开发经验
  - `理财/` - 投资理财知识（非交易记录）
  - `工作/` - 工作相关技能和经验
- **特点**：长期维护、持续更新

### 资源笔记 `/03-Resources/`
- **存放内容**：参考资料、外部资源、读书笔记
- **触发条件**：整理自外部文章、文档、书籍
- **命名格式**：`资源名称.md`
- **Frontmatter**：
  ```yaml
  ---
  title: 标题
  source: 来源链接
  date: YYYY-MM-DD
  tags: [标签1, 标签2]
  ---
  ```

### 模板 `/Templates/`
- **Book Note.md** - 读书笔记模板
- **Daily Note.md** - 日记模板
- **Project Note.md** - 项目笔记模板

## 快速决策流程

```
新笔记内容
    │
    ├─ 是否包含股市/交易数据？
    │   └─ 是 → /股市日记/
    │
    ├─ 来源是否为视频/直播？
    │   └─ 是 → /Videos/
    │
    ├─ 是否有明确项目目标和截止日期？
    │   └─ 是 → /Projects/
    │
    ├─ 是否为持续维护的主题（编程/理财/工作）？
    │   └─ 是 → /02-Areas/对应领域/
    │
    └─ 是否为外部参考资料/读书笔记？
        └─ 是 → /03-Resources/
```
