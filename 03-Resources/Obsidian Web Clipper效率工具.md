## 🔧 功能介绍

### 1. 剪藏插件
- 将网页内容按指定格式提取
- 保存到 Obsidian 指定位置
- 输出格式：本地 Markdown 文档

### 2. AI 功能
- 填入 API Key、模型 ID 即可启用
- 支持 OpenRouter 调用各种模型
- 推荐模型：Gemini 3.1 Flash（便宜、上下文大）

### 3. YouTube 字幕提取（新功能）
- 更新后默认支持完整字幕提取
- 无需额外配置
- 一两秒即可完成

## 💡 使用场景

### 场景1：剪藏公众号文章
- 让 AI 做总结和提炼

### 场景2：YouTube 视频处理
- 提取完整字幕
