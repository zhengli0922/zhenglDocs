# 保姆级AgentSkill封装教程，学会让工作效率飙升10倍！

## 核心问题

使用AI的两大痛点：
1. **每次都要重新提要求** — 同样的工作今天做一遍明天还要再来
2. **产出不稳定** — 同样的操作结果每次不一样，用AI像赌运气

根因：没有给AI建立标准化的工作体系。

## Skill是什么

Skill = 给AI建一套标准化工作体系，让重复性工作自动执行。

### Skill效果演示

1. **发票报销整理**：扔一堆发票图片，自动识别金额/日期/类别，输出标准报销表格
2. **草图转网页**：画个粗糙草图，不给任何提示，AI自动生成前端网页
3. **文档转PPT**：丢一份文字报告，自动生成带品牌色/字体/排版的PPT

共同点：全程不给额外提示，产出稳定，像拥有训练好的员工。

## Skill文件夹结构

以麦当劳做汉堡为比喻：

| 文件夹 | 比喻 | 作用 |
|--------|------|------|
| `SKILL.md` | 员工手册 | 告诉AI职责、触发场景、执行步骤、输出标准 |
| `reference/` | 严格菜谱 | AI必须遵守的标准格式、内容规范、行业规则 |
| `scripts/` | 自动化设备 | 可直接运行的脚本，处理机械步骤 |
| `assets/` | 预制原料 | 每次输出用到的模板文件（如PPT模板） |

## 如何创建Skill

### 方法一：从已磨合好的对话提取

当跟AI来回调教出满意结果后，**不要关掉对话框**，发送：

> 我们刚才的对话已经磨合出了完整的工作流和输出标准。现在请你将这个过程整理成一个标准的AgentSkill，要求如下：
> 1. 创建完整的Skill文件夹结构
> 2. SKILL.md要写清楚职责、触发场景、执行步骤和输出标准
> 3. reference放入确认过的格式要求和内容标准
> 4. 可自动化的步骤写入scripts
> 5. assets文件夹放入重复使用的模板文件
> 最后输出一个可以直接安装使用的Skill文件夹

### 方法二：从GitHub搜索现成Skill

- 搜索别人做好的Skill下载导入
- 不满意就让AI根据自己习惯优化迭代
- 别人80分的Skill → 优化成你专属的100分

## 推荐工具
