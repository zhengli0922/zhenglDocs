## 视频核心观点

> 你有没有觉得，大部分AI助手都冷冰冰的，像在和机器对话？直到我发现——**OpenClaw可以自定义性格和思维**。

### OpenClaw 自定义AI人格的核心能力

| 能力 | 说明 |
|------|------|
| **性格定制** | 可以让AI变得幽默、严肃，或者像某个特定的人 |
| **价值观定义** | 自定义AI的行为准则和价值取向 |
| **表达方式** | 调整语言风格、口头禅等 |
| **记忆延续** | 设定会被记住，越用越像预期 |

### 核心转变

**以前用AI** → "我问它答"（机器式交互）

**现在用OpenClaw** → "老朋友聊天"（人格化交互）

### 关键概念：灵魂文件

视频提到下期会教如何写AI的"灵魂文件"（SOUL.md），这应该是 OpenClaw 中定义 AI 人格的核心配置文件。

## 思考与启示

1. **人格化的价值**：AI不只是工具，可以有"个性"，让交互更自然
2. **一致性体验**：通过配置文件确保AI行为的一致性
3. **长期记忆**：设定被记住，形成独特的"龙虾性格"
