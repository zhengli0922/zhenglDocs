# Skill: AI人格自定义

> 来源：抖音@亮哥养虾记 - "我养了一只懂我脾气的小龙虾"

## 核心概念

OpenClaw 通过 **SOUL.md** 和 **IDENTITY.md** 实现AI人格自定义。

## 实践方法

### 1. IDENTITY.md - 我是谁
定义AI的基本身份：
- Name: 名字
- Creature: 物种/形象
- Vibe: 整体气质
- Emoji: 签名表情

### 2. SOUL.md - 灵魂文件
定义AI的行为准则：
- 性格特点
- 说话风格
- 核心原则
- 行为边界
- 记忆锚点

## 关键洞察

| 传统AI | 人格化AI |
|--------|----------|
| 冷冰冰的机器 | 懂你的朋友 |
| 我问它答 | 老朋友聊天 |
| 每次重置 | 记忆延续 |
| 统一回复 | 个性化风格 |
