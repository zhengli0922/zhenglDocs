# 给你的OpenClaw装上全网搜索，Agent-Reach开源免费，全平台适配

OpenClaw装好之后，要么不能获取实时信息，要么需要面对Tavily Web Search的高昂API费用。今天分享开源神器 **Agent-Reach**，专门解决OpenClaw的"断网难题"，无需付费 API Key，完全开源免费。

## Agent-Reach 是什么？

专为 AI Agent 打造的全网数据访问扩展工具，让 AI 自由访问、读取、解析全网各大主流平台的内容，解决原生 OpenClaw 联网能力弱、适配平台少、使用成本高的痛点。

## 核心优势

- **完全开源免费**：所有核心工具开源，基础 API 免费，无需付费 API Key
- **全平台覆盖**：国内外主流平台全覆盖（通用网页、视频、社交、技术社区、资讯）
- **操作门槛低**：AI 自动完成安装配置，无需手动写代码
- **隐私安全**：登录信息仅存储本地，代码开源可审计
- **兼容性强**：支持 Claude Code、OpenClaw、Cursor、Windsurf 等主流 Agent

## 安装步骤

1. **前期准备**：确保 AI Agent 支持命令行操作
2. **一键安装**：将项目地址给 AI Agent，输入"帮我安装这个 Agent-Reach 工具"，AI 自动完成全流程（1-3分钟）
3. **诊断启用**：命令行输入 `agent-reach doctor`，运行全链路诊断
